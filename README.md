# CepTur
React Native programlama kullanarak geliştirmiş olduğum, İstanbul'a gelen turistler için gezebilecekleri yerleri, güzel restoranları, tarihi yerleri konum ve açıklamlarıyla sunan bir gezi rehberi uygulamasıdır.


## Uygulamanın Barındırdığı İçerikler
- Şehirdeki gezilebilecek yerleri listelemektedir.Bunlar hakkında kısa bilgiler vermektedir.
- Kullanıcıların yapmış olduğu yorumları gösterir.
- Kullanıcıların bu yerler için vermiş olduğu reyting oranını gösterir.
- Kullanıcıların çekmiş olduğu fotoğrafları barındırır.
## Kullanılan Teknolojiler
### Firebase
Uygulamada yer alan giriş ve kayıt olma Firebase authentication, özellikleri Firestore kullanılarak yapılmıştır.
## Kullanılan API Servisi
### Google Places Api
Uygulamadaki bütün mekanların bilgileri, fotoğrafları, yorumları, reyting oranları, bulunduğu konumu Google Places API kullanılarak oluşturulmuştur. Kullanıcıların anlık hareket bilgileri kullanılmaktadır.
